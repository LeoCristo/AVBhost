Detecção de Queimadas — Rápido

Requisitos
- Python 3.8+
- pip

Instalação
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1  # PowerShell
python -m pip install -r requirements.txt
```

Executar
```powershell
python fire_detector_app.py
```
